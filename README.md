# 📚 Study Planner Web App

A simple yet powerful **Task & Study Planner** built with **HTML, CSS, and JavaScript**.  
This project showcases my front-end development skills, including **DOM manipulation**, **local storage**, and **UI design**.

🔗 **Live Demo:** [GitHub Pages Link](https://msamin-25.github.io/study-planner)

---

## ✨ Features

- ✅ Add, mark complete, and delete tasks
- 📊 Progress bar that updates as tasks are completed
- ⏳ Built-in Pomodoro Timer (25 min)
- 🌙 Dark / Light mode toggle
- 💾 Saves tasks in **Local Storage**
